# Task Managing & Tracking App Built for Tophat

## Features
* Create Task
* Edit Task
* Delete Task
* Start/Stop Task
* User Authentication

## Tech Stack Used - MERN Stack
* MongoDB
* ExpressJs
* ReactJs
* NodeJs
* MUI
* Redux
* JWT

## Startup
### Server
```
  cd server/
```
```
  pnpm nodemon
```

### Client
```
  cd client/
```
```
  pnpm dev
```
